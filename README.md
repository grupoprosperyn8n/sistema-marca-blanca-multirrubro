# SISTEMA MARCA BLANCA MULTIRRUBRO

> Demo piloto: Gestión de Salones de Belleza

Sistema modular multirrubro con inteligencia artificial. Permite gestionar negocios de cualquier rubro (belleza, retail, servicios, gastronomía) con una misma plataforma marca blanca. El vertical de belleza es la demo inicial.
## Stack
Supabase | Airtable | n8n | Surge.sh | GitHub

## Inicio rapido
Abre AGENTS.md y elige tu agente. START.md es historia (compatibilidad).
Luego corre: `bash exportar-proyecto.sh /ruta/proyecto /ruta/fabrica` para instalar Factory Harness v3.7.2 en la raiz.
